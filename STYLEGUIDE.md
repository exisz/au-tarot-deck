# STYLEGUIDE.md — Arcana Deck

## Brand Identity
- **Name:** Arcana Deck
- **Tagline:** Online Tarot with Deck Building & Journal
- **Tone:** Mystical yet accessible. Inviting, not intimidating.

## Visual Design
- **Theme:** Deep purple + gold/amber accent
- **Primary palette:** `#1a0a2e` (bg), `#7c3aed` (purple-600), `#d4a017` (gold)
- **Card imagery:** Rider-Waite-Smith public domain (1909)
- **Typography:** System fonts, clean sans-serif
- **Icons:** Emoji-based for suits, RWS images for card detail pages

## Competitive Reference
- **labyrinthos.co** — Best-in-class content depth, lesson structure, beautiful illustrations
- **biddytarot.com** — SEO authority, comprehensive meanings, strong community
- **Our differentiator:** Balatro-style deck building + personal reading journal — no competitor offers this combination

## Content Standards
- Major Arcana: 300+ word descriptions with symbolism, imagery, reading guidance
- Minor Arcana: Element + suit themes (to be expanded in future iterations)
- Bilingual: English primary, Chinese (中文) secondary labels

## UI Patterns
- Cards display with RWS artwork (Major Arcana) or suit emoji (Minor Arcana)
- Purple-gold color scheme consistent across all pages
- Mobile-first responsive layout
- DaisyUI components where applicable

## SEO
- URL pattern: `/card/{slug}` (e.g., `/card/the-fool`)
- Each card has JSON-LD Article schema
- Keywords include both English and Chinese card names
